Installation Guide:-
1. Install Java 7+(https://www.oracle.com/java/technologies/javase-downloads.html)
2. Install mysql-server, mysql-connector(https://dev.mysql.com/downloads/installer/) User Name : root, Password : root
3. Run script.sql file in mysql client.
4. Install postman to run the APIs. (Can be added from chrome extension)
5. Create account on GitHub and install git client. Login with your credentials.
6. Clone the project from "https://github.com/ainajain1/MeetingRequest.git" using git clone command.\


Build Guide:-
1. Run jar file using command "java -jar MeetingRoomBooking-1.0.jar" from "target" folder. Server will be running on "localhost:8080".
2. Check ppt "Online Meeting Room Booking.pptx" for API request response and URL.
3. Run the APIs on postman to verify the response.
